# Web Framework & Server
fastapi - The core backend framework for routing and background tasks.
uvicorn - The ASGI server used to run the FastAPI application.

# Database & Validation
sqlalchemy - The ORM (Object Relational Mapper) used for database models.
pydantic - Used for data validation and standardizing raw JSON from APIs.
pydantic-settings - Manages environment variables and thresholds from the .env file.
python-dotenv - Loads environment variables from the .env file.

# HTTP Client (Async)
httpx - An asynchronous HTTP client used for concurrent network requests.

# Frontend Dashboard
streamlit - The framework used to build the 3-page monitoring dashboard.

# Testing Tools
pytest - The primary testing framework for the required unit tests.
pytest-asyncio - Allows pytest to run the asynchronous code used in the backend.
respx - Used for mocking external API responses (Weather, Finance, Geological).

# Logging
loguru - A library for professional-grade logging (INFO, WARNING, ERROR).
